DriftGate

Catch breaking API changes before consumers do.

DriftGate combines contract governance, webhook reliability, schema drift evidence, payload history, and guided review into one local-first platform. Run it in Docker, wire it to Azure Service Bus or Cosmos-compatible storage when you need them, and keep the whole workflow explainable.
